# ICW — Indonesia Corruption Watch
**Organization:** ICW (NGO)
**Portal:** https://antikorupsi.org | https://tracker.antikorupsi.org
**API type:** ⚠️ Web database (scraping)

## Overview
Tracked corruption cases, KPK indictments, procurement corruption. The prosecution tracker is a valuable resource for due diligence.

## Key Resources
- Corruption tracker: `tracker.antikorupsi.org`
- Procurement watch: `pengadaan.id`
- Investigation reports and analysis

## Gotchas
1. Web scraping only — no API
2. Valuable for background checks and due diligence
3. Independent NGO data — not government official
