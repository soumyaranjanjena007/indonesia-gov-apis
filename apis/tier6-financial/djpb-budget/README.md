# DJPB — Budget Execution Data
**Agency:** Direktorat Jenderal Perbendaharaan, Kementerian Keuangan
**Portal:** https://djpb.kemenkeu.go.id/portal/id/data/apbn-realisasi
**API type:** ⚠️ XLS/CSV downloads

## Overview
Real-time APBN spending execution by ministry, function, program. Monthly reports.

## Key Data
- APBN revenue and expenditure realization
- Ministry-level budget breakdown
- Monthly execution reports
- Also: `monev.anggaran.kemenkeu.go.id` for granular views

## Gotchas
1. Excel downloads available
2. Monthly publication cycle
3. Combine with BPS data for budget-vs-outcome analysis
