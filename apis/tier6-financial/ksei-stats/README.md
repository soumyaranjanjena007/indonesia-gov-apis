# KSEI — Securities Investor Statistics
**Agency:** Kustodian Sentral Efek Indonesia
**Portal:** https://ksei.co.id/publikasi/statistik
**API type:** ⚠️ PDF/XLSX monthly stats

## Overview
Investor count by province, sub-registry type, product (SBN, saham, reksadana). Aggregate market statistics, not individual position data.

## Key Data
- Monthly investor count reports
- Breakdown by province and product type
- Sub-registry statistics

## Gotchas
1. Monthly data file downloads (XLSX/PDF)
2. Aggregate stats only — no individual investor data
3. Useful for understanding investment penetration across Indonesia
