# OGP Indonesia — Open Government Partnership
**Organization:** Open Government Indonesia
**Portal:** https://opengovindonesia.org
**API type:** ⚠️ HTML reports

## Overview
OGP national action plan tracking, ministry transparency scores, open data implementation status. Useful for understanding which ministries are data-mature.

## Gotchas
1. Periodic reports on government transparency
2. Tracks which ministries have open data commitments
3. Useful meta-source for prioritizing data collection targets
